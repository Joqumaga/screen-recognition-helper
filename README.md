# 屏幕识别点击助手

一个 Windows 桌面游戏辅助工具，能够**框选屏幕区域 → OCR 识别文字 → 匹配目标 → 自动鼠标点击**。

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 多区域监控 | 拖拽框选一个或多个屏幕区域，独立启用/禁用 |
| OCR 文字识别 | 支持中文简体 + 英文混合识别 |
| 目标匹配 | 设定目标文字，支持模糊匹配（包含匹配） |
| 自动点击 | 匹配成功后将鼠标移动到文字中心自动点击 |
| 可调参数 | 点击范围、点击延时、扫描间隔均可自由调节 |
| 专属目标 | 每个监控区域可设专属目标文字，互不干扰 |
| 运行日志 | 实时显示每次扫描的识别和点击记录 |
| 浮动窗模式 | 最小化后自动切换浮动指示窗，不遮挡屏幕 |

## 安装说明

### 方式一：直接运行 Python 源码

**1. 安装 Python 3.8+**

从 [python.org](https://www.python.org/downloads/) 下载安装，安装时勾选 **"Add Python to PATH"**。

**2. 安装 Tesseract OCR**（必需）

```bash
# 方法 A：使用 winget（推荐）
winget install "UB-Mannheim.TesseractOCR"

# 方法 B：手动下载安装
# 从 https://github.com/UB-Mannheim/tesseract/wiki 下载安装包
```

默认安装路径：`C:\Program Files\Tesseract-OCR\tesseract.exe`

> **中文识别支持**：需要安装中文简体语言包。将 `chi_sim.traineddata` 放入 `C:\Program Files\Tesseract-OCR\tessdata\` 目录。
> 可从 [tessdata_fast](https://github.com/tesseract-ocr/tessdata_fast/raw/main/chi_sim.traineddata) 下载（约 2.4MB）。

**3. 安装 Python 依赖**

```bash
# 在项目目录下执行
pip install -r requirements.txt
```

**4. 启动软件**

```bash
python src/main.py
```

### 方式二：使用打包好的 .exe

1. 从 [Releases](../../releases) 页面下载最新版 `屏幕识别点击助手.exe`
2. 先安装 Tesseract OCR（同上步骤 2）
3. 双击运行 `.exe` 文件

## 使用指南

### 第一步：添加监控区域

1. 点击左侧 **「添加区域」** 按钮
2. 屏幕变半透明，**按住鼠标左键拖拽** 框选要监控的区域
3. 松开鼠标确认，按 **ESC** 取消
4. 可以添加多个区域，每个区域用开关独立启用/禁用
5. 每个区域可展开专属目标面板，设置该区域专属的匹配文字

### 第二步：设置目标文字

1. 在右侧 **「目标文字」** 输入框中输入要自动点击的文字
2. 点击 **「添加」** 或按 **回车键** 加入目标列表
3. 支持多个目标，点击 x 可删除

### 第三步：测试识别（可选）

1. 在 **「识别预览」** 中选择一个区域
2. 点击 **「测试识别」** 查看该区域的文字识别效果
3. 匹配到的目标会以绿色高亮显示

### 第四步：开始监控

1. 调节 **「扫描间隔」** 控制识别频率（默认 200ms）
2. 调节 **「点击范围」** 控制点击随机偏移（0-20px）
3. 调节 **「点击延时」** 控制点击前等待时间（0-500ms）
4. 点击 **「▶ 开始监控」** 启动自动识别与点击
5. 底部日志面板会实时显示运行状态

### 第五步：停止监控

- 点击 **「■ 停止监控」** 即可暂停
- 直接关闭窗口也会自动停止

## 配置项说明

| 配置 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 扫描间隔 | 50-2000ms | 200ms | 每次扫描之间的间隔 |
| 点击范围 | 0-20px | 5px | 点击位置随机偏移半径 |
| 点击延时 | 0-500ms | 50ms | 鼠标移动到点击之间的等待 |

## 注意事项

- ⚠️ 识别语言支持中文简体 + 英文，需确保已安装对应 Tesseract 语言包
- ⚠️ 运行期间鼠标会被自动控制，按 `Win+D` 或 `Ctrl+Alt+Del` 可中断
- ⚠️ 建议配合窗口化游戏使用，不要遮挡目标区域
- ⚠️ 首次使用请在「识别预览」中测试识别效果，确保 OCR 正常工作
- 💡 扫描间隔建议 200-500ms，过快会增加 CPU 占用
- 💡 监控启动后窗口会自动最小化为浮动窗，避免遮挡屏幕

## 技术栈

| 组件 | 技术 | 用途 |
|------|------|------|
| 界面 | CustomTkinter | 现代化桌面 UI |
| OCR | Tesseract 5.4.0 | 文字识别 |
| 截图 | mss (GDI) | 高速屏幕截图 |
| 图像处理 | OpenCV + Pillow | 预处理提升识别率 |
| 鼠标控制 | PyAutoGUI | 模拟鼠标点击 |
| 打包 | PyInstaller | 生成单个 .exe |

## 项目结构

```
src/
├── main.py               # 程序入口
├── config.py              # 全局配置
├── config.json            # 区域&目标持久化配置
├── ui/
│   ├── app_window.py      # 主窗口（含 OCR 预览、目标管理、区域管理）
│   ├── region_selector.py # 区域框选（全屏半透明拖拽）
│   ├── floating_window.py # 监控浮动指示窗（最小化后显示）
│   ├── log_panel.py       # 日志面板（带级别颜色和时间戳）
│   └── tooltip.py         # 悬停提示工具
├── core/
│   ├── monitor.py         # 监控主循环（5 步分步异常捕获）
│   ├── screen_capture.py  # 截图（mss 线程安全封装）
│   ├── ocr_engine.py      # OCR 识别（中英文混合）
│   ├── matcher.py         # 目标匹配（精确 + 模糊两种模式）
│   └── clicker.py         # 鼠标点击（随机偏移 + 可调延时）
└── utils/
    └── image_processor.py # 图像预处理（灰度→二值化→放大→去噪）

docs/                      # 技术文档
dev-logs/                  # 开发日志（按日期）
build.bat                  # Windows 打包脚本
build.spec                 # PyInstaller 打包配置
```
